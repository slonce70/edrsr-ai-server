### 👨‍💼 Добавление новых администраторов

**Вариант 1: Через админскую панель**
1. Пользователь регистрируется в расширении
2. В админке: Пользователи → Найти → "Сделать админом"

**Вариант 2: Через командную строку**
```bash
# Если знаете User ID из базы
npm run admin:grant <user_id>

# Если есть Service Role Key для Supabase  
npm run admin:create <email>
```

### ❗ Важные заметки

1. **Первый вход:** Используйте тот же email/пароль что и в расширении
2. **Забыли пароль:** Сбросьте пароль через расширение
3. **Проблемы с доступом:** Проверьте что права администратора назначены
4. **Безопасность:** Не предоставляйте права администратора случайным пользователям

### 🛠️ Техническая информация

**API Endpoints:**
- `GET /api/admin/dashboard` - Дашборд  
- `GET /api/admin/users` - Пользователи
- `GET /api/admin/jobs` - Задания
- `POST /api/admin/system/cleanup` - Очистка
- `GET /api/admin/audit-log` - Аудит

**Файлы:**
- `/admin` - Веб-интерфейс
- `/api/admin/*` - API для админки
- `server/routes/admin.js` - Backend логика
- `server/middleware/adminAuth.js` - Проверка прав
