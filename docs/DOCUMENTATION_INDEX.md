## **🚀 Быстрый старт**

### **Минимальный набор для начала работы:**
```bash
# 1. Изучить быстрые команды
cat QUICK_COMMANDS.md

# 2. Настроить переменные окружения
cp server/.env.example server/.env
# Отредактировать server/.env

# 3. Запустить проект
npm run dev

# 4. Открыть админку
# http://localhost:4000/admin
```

### **Для администраторов:**
```bash
# 1. Создать администратора
cd server
npm run admin:grant

# 2. Открыть админку
# http://localhost:4000/admin
# Логин: ваш_email
```
